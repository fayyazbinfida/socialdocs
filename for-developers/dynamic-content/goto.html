# Goto

goto property of server response is optional.

# Goto Step / Sub Flow

Use this response to goto a step or sub flow:

{
    "version": "v1",
    "content": {
        "goto": "SOMETHING"
    }
}

You can goto:

Goto SOMETHING = e.g.
A step step name (unique) Welcome
A step node_ns f244n7831
A sub flow sub_flow_ns f244s1032
